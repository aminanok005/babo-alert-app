# Project Memory: Al-Anbiya 7-Day Series

## 1. Project Goal
ผลิตวิดีโอซีรีส์ 7 ตอน จากซูเราะฮ์ อัล-อันบิยาอ์ อายะฮ์ 1-16 เพื่อลงต่อเนื่อง 7 วัน วันละ 8 นาที

## 2. Decided Structure (8-Minute Timeline)
- **0:00-0:15:** Hook (15 วินาที)
- **0:15-2:00:** ปูพื้นฐานและปัญหา
- **2:00-6:30:** เนื้อหาหลัก (ตัฟซีรและการปรับใช้)
- **6:30-8:00:** สรุปบทเรียนและ CTA สำหรับตอนถัดไป

## 3. Episode Titles & Hooks (Ref: Previous Session)
1. **Day 1:** ความเพิกเฉย (อายะฮ์ 1-3) - "การสอบสวนใกล้เข้ามาแล้ว..."
2. **Day 2:** ความลับของใจ (อายะฮ์ 4) - "อัลลอฮ์ทรงรู้ทุกคำพูด..."
3. **Day 3:** กำแพงแห่งข้ออ้าง (อายะฮ์ 5-6) - "ทำไมความจริงถึงกลายเป็นฝัน..."
4. **Day 4:** แบบอย่างที่เป็นมนุษย์ (อายะฮ์ 7-9) - "นบีก็คือคนธรรมดา..."
5. **Day 5:** คัมภีร์แห่งเกียรติยศ (อายะฮ์ 10) - "ยกเกียรติด้วยอัลกุรอ่าน..."
6. **Day 6:** บทเรียนจากซากปรักหักพัง (อายะฮ์ 11-15) - "ความอธรรมที่มีราคาจ่าย..."
7. **Day 7:** โลกไม่ใช่สนามเด็กเล่น (อายะฮ์ 16) - "ใช้ชีวิตอย่างมีเป้าหมาย..."

## 4. Content Style
- **Tone:** จริงจัง, ปลุกใจ, มีเมตตา
- **Editing Style:** เน้นความสะอาดตา, ใช้ข้อความอายะฮ์ภาษาอาหรับพร้อมคำแปลขึ้นหน้าจออย่างชัดเจน
- **Language:** ภาษาไทยเข้าใจง่าย ไม่ใช้คำอาหรับยากโดยไม่จำเป็น
- **Target Audience:** มุสลิมไทยทั่วไป, วัยรุ่น-วัยทำงาน, ผู้สนใจอิสลาม

## 5. Technical Standards
- **Video Format:** 1080x1920 (Vertical 9:16)
- **Duration:** 8 นาที ±30 วินาที
- **Audio:** Edge-TTS Voice th-TH-PremwadeeNeural
- **Font:** IBM Plex Sans Thai
- **Output:** MP4 H.264 + AAC
