# Knowledge Base: Surah Al-Anbiya (1-16) & Production Rules

## 1. Quranic Framework: Surah Al-Anbiya 1-16

### General Theme
การตักเตือนถึงวันสอบสวนและความเพิกเฉยของมนุษย์

### Key Verses Breakdown

| อายะฮ์ | เนื้อหาหลัก | คำสำคัญ |
|--------|------------|---------|
| 1-3 | ความประมาท (Ghaflah) ของมนุษย์ต่อวันกิยามะฮ์ | حساب, غفلة, معرضون |
| 4 | ความรอบรู้ของอัลลอฮ์ต่อทุกคำพูด | علم, سر, علانية |
| 5-6 | ข้ออ้างและการปฏิเสธความจริงของกลุ่มชนในอดีต | كذب, ظلم, قرآن |
| 7-9 | นบีคือมนุษย์ที่เป็นแบบอย่าง และจุดจบของผู้ฝ่าฝืน | بشر, رسول, رحمة |
| 10 | เกียรติยศของมนุษย์อยู่ในข้อเตือนใจ (อัลกุรอ่าน) | ذكر, شرف, فقه |
| 11-15 | บทเรียนจากเมืองที่ถูกทำลายเพราะความอธรรม | قرية, ظلم, هلك |
| 16 | เป้าหมายของการสร้าง (โลกนี้ไม่ใช่เรื่องเล่นๆ) | لعب, عبادة, هدف |

## 2. Tafsir Principles (หลักการตัฟซีร)

### Authenticity
- อ้างอิงแหล่งตัฟซีรที่เชื่อถือได้ (เช่น Ibn Kathir, Al-Jalalayn, Tafsir Al-Muyassar)
- ไม่ตีความเองโดยไม่มีหลักฐาน

### Context (Asbab al-Nuzul)
- ต้องระบุบริบทของการประทานอายะฮ์เพื่อป้องกันการตีความผิด
- เชื่อมโยงกับเหตุการณ์สมัยท่านนบีเมื่อเหมาะสม

### Balanced Message
- นำเสนอทั้งความหวัง (Targheeb) และการตักเตือน (Tarheeb)
- หลีกเลี่ยงการทำให้ผู้ฟังท้อแท้หรือกลัวเกินไป

## 3. Video Production Standards

| Standard | Value |
|----------|-------|
| Duration | 8 นาทีต่อคลิป |
| Structure | 1 สคริปต์หลัก แตกเป็น 7 วิดีโอสั้น |
| Engagement | Pattern Interrupt ทุก 45-60 วินาที |
| Visual | ข้อความอาหรับ + คำแปลไทยชัดเจน |
| Audio | เสียงไทยธรรมชาติ ไม่เร็วจนเกินไป |

## 4. Recommended Tafsir Sources

### Primary (ง่าย เข้าใจเร็ว)
- **Tafsir Al-Muyassar** - ภาษาง่ายที่สุดสำหรับคนทั่วไป
- **Tafsir Al-Jalalayn** - กระชับ ได้ใจความ

### Secondary (ลึก มีเรื่องราว)
- **Tafsir Ibn Kathir** - มีเรื่องราวประวัติศาสตร์ครบถ้วน
- **Tafhim al-Quran (Maududi)** - เชื่อมโยงกับชีวิตปัจจุบัน

### Thai Sources
- **Quran.com** - คำแปลภาษาไทย อ.อิบรอฮิม ภูมิ
- **YouTube:** อ.อิสมาอีล ลุตฟี จะปะกียา, อ.สมชาย พูลสวัสดิ์
